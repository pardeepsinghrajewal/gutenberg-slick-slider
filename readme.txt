Image Slick Slider for Gutenberg

== Description ==

This is a slick image slider. Please note that it does not support videos.

== Installation ==


1. Upload the plugin files to the `/wp-content/plugins/gutenberg-slick-slider` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
